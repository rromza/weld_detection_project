# Weld Quality Classifier
Проект для классификации качества сварных швов с использованием сверточной нейросети.

# Возможности
Классификация сварных швов:

✅ good_weld - Качественный шов

❌ bad_weld - Некачественный шов с дефектами

⚠️ no_weld - Шов не обнаружен

Интерактивный интерфейс через Streamlit

Подробная визуализация вероятностей предсказаний


# Архитектура модели
Основа: EfficientNet-B1 (предобученная)

Классификатор: Трехслойный полносвязный с Dropout

Входной размер: 224×224 пикселей

Количество классов: 3
